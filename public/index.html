Basic Usage:

Characters

Get with meta_data and characters properties 
{
  characters{
    character_data{
      name
      height
      gender
    }
    meta_data{
      total_height{
        cm
        inches
        feet
      }
      total_characters
    }

  }
}
Get characters by sort 

{
  characters(sort: height_asc){
    character_data{
      name
      height
    }
  }
}
Get characters by gender filter 
{
  characters(gender: male){
    character_data{
      name
      gender
      height
    }
  }
}
You can use both sort and filter characters(gender: male, sort: height_asc){ ... }

Movies

Get movies 
{
  movies{

    episode_id
    comment_count
  }

}
